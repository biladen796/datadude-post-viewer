# Twitter

Twitter is a node extension that allows you to add an Twitter to your editor.

- [react-tweet](https://www.npmjs.com/package/react-tweet)

## Usage

```tsx
import { Twitter } from 'reactjs-tiptap-editor'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
  Twitter
];
```
